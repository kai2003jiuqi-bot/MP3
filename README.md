# 需求

1. 做一个 MP3
2. 功能：
   - 播放音乐
   - 暂停、上下首、进度条
   - 歌单管理：添加歌曲、删除歌曲、改变歌曲顺序
   - 电脑通过 USB 传输 MP3 文件
   - 多语言歌名显示
   - 显示图片
   - 模式：随机播放、顺序播放、单曲循环
   - 保存播放列表

# 总体方案

- **主控**：STM32F405RGT6
- **MP3 解码芯片**：VS1003
- **显示芯片**：ST7789 - 240×320
- **触摸芯片**：XPT2056 - 240×320
- **外部 Flash**：W25Q64、SD 卡

## UI 设计

- **播放界面**：图片显示、暂停按钮、上下首按钮、进度条、模式按钮
- **歌单界面**：三行歌名显示，一个选中框，每行歌名右边是状态图标，底部三个按钮，分别是上、确定、下
- **歌曲库界面**：三行歌名显示，一个选中框，每行歌名右边是状态图标，底部三个按钮，分别是上、确定、下
- **文件传输界面**：一个图标

## 整体工作流程

1. 上电 → 外设初始化 → 驱动初始化 → 播放界面
2. **播放界面**：
   - 按下上下首切换歌曲，同时保存歌单到 Flash
   - 按下暂停按钮暂停，再次按下继续播放
   - 按下模式按钮切换模式
   - 拖动进度条可以改变播放进度
3. **歌单界面**：
   - 按上下键移动歌曲列表，按下确定，删除歌曲
   - 长按确定，选中框变绿，再按下确定，选中框颜色恢复改变颜色，歌曲顺序改变，保存歌单到 W25Q64
4. **歌曲库界面**：
   - 按上下键移动歌曲列表，按下确定，添加歌曲到歌单
5. 长按增大音量键向右切换界面、长按减小音量键向左切换界面、点击音量键增大或减小音量
6. 电脑插上 USB，音乐暂停，进入文件传输界面，拔出 USB，进入播放界面

# 硬件架构

- 主控 — SPI — VS1003
- 主控 — SPI — ST7789
- 主控 — SPI — XPT2056
- 主控 — SPI — W25Q64
- 主控 — SDIO — SD 卡
- 主控 — USB — 电脑

# 使用方法

- 该项目基于 STM32CubeMX 生成工程
- 使用 VSCODE 的 IEDE 插件或者 Keil 进行编译和下载
