# MP3 Player — 嵌入式音乐播放器

[![License](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)
[![Platform](https://img.shields.io/badge/Platform-STM32F405RGT6-green)]()
[![Status](https://img.shields.io/badge/Status-开发中-yellow)]()

基于 **STM32F405RGT6** 的嵌入式 MP3 播放器，采用硬件解码，支持触摸 GUI、USB 大容量存储和歌单管理。

---

## 📋 功能特性

- 🎵 **音乐播放** — 播放、暂停、上下首切换、进度条拖拽
- 📂 **歌单管理** — 添加/删除歌曲、改变播放顺序、自动保存到 Flash
- 🔁 **播放模式** — 随机播放、顺序播放、单曲循环
- 🖥️ **触摸屏 UI** — 240×320 TFT 彩屏 + 触摸操作
- 🌐 **多语言支持** — 正确显示多语言歌名
- 🖼️ **专辑封面** — 支持图片显示
- 💾 **USB 传输** — 连接电脑即可传输 MP3 文件
- 💾 **持久化存储** — 歌单和设置保存到外部 Flash

---

## 🧱 硬件架构

| 组件             | 型号             | 接口   | 说明                         |
|------------------|------------------|--------|------------------------------|
| **主控 MCU**     | STM32F405RGT6    | —      | ARM Cortex-M4, 168 MHz       |
| **MP3 解码芯片** | VS1003           | SPI    | 硬件解码 MP3                 |
| **显示屏**       | ST7789           | SPI    | 240×320 TFT                  |
| **触摸芯片**     | XPT2046          | SPI    | 电阻式触摸                   |
| **外部 Flash**   | W25Q64           | SPI    | 8 MB — 用于歌单和数据存储    |
| **存储卡**       | SD 卡            | SDIO   | 存放音乐文件                 |
| **电脑连接**     | USB              | USB    | 大容量存储设备模式           |

### 连接框图

```
                   ┌─────────────┐
                   │    USB      │ ←→ PC（大容量存储）
                   └──────┬──────┘
                          │
┌──────────┐  SPI  ┌──────┴──────────┐  SPI  ┌──────────────┐
│ VS1003   │←──────→│                 │←──────→│  ST7789      │
│ (解码器)  │       │  STM32F405RGT6  │       │  (显示屏)    │
└──────────┘       │                 │       └──────────────┘
                    │                 │  SPI  ┌──────────────┐
┌──────────┐  SDIO  │                 │←──────→│  XPT2046     │
│ SD 卡    │←──────→│                 │       │  (触摸)      │
└──────────┘       └──────┬──────────┘       └──────────────┘
                          │  SPI
                   ┌──────┴──────┐
                   │  W25Q64     │
                   │  (Flash)    │
                   └─────────────┘
```

## 💻 软件架构

### 界面说明

| 界面           | 说明                                           |
|----------------|------------------------------------------------|
| **播放界面**   | 专辑封面、暂停/播放、上下首、进度条、模式切换   |
| **歌单界面**   | 三行歌名列表 + 选中框 + 状态图标、上下/确定按键 |
| **歌曲库界面** | 浏览 SD 卡歌曲、添加到歌单                      |
| **文件传输**   | USB 连接时进入，播放自动暂停                    |

### 工作流程

```
上电
 │
 ├── 外设初始化（时钟、GPIO、SPI、SDIO、USB）
 ├── 驱动初始化（VS1003、ST7789、XPT2046、W25Q64、SD 卡）
 │
 └── 进入播放界面
        │
        ├── [▲/▼ 短按]        → 切换上下曲，自动保存歌单到 Flash
        ├── [▲/▼ 长按]        → 向右/向左切换界面
        ├── [暂停键]           → 暂停/继续播放
        ├── [模式键]           → 随机 ↔ 顺序 ↔ 单曲循环
        ├── [进度条]           → 拖拽改变播放进度
        ├── [USB 插入]         → 暂停播放 → 进入文件传输界面
        └── [USB 拔出]         → 恢复播放 → 回到播放界面
```

### 歌单管理操作

| 操作             | 方式                                          |
|------------------|-----------------------------------------------|
| **浏览歌单**     | 短按上/下滚动列表                             |
| **删除歌曲**     | 短按确定选中歌曲                              |
| **改变顺序**     | 长按确定（选中框变绿）→ 上/下移动 → 短按确定放置 |
| **添加歌曲**     | 进入歌曲库 → 浏览 → 短按确定添加              |
| **自动保存**     | 任何修改自动写入 W25Q64                       |

## 🚀 快速开始

### 环境要求

- [STM32CubeMX] — 工程生成工具
- [VS Code] + [IAR/Keil 插件]，**或** 直接使用 [Keil MDK]

### 构建与烧录

1. **生成工程** — 使用 STM32CubeMX 打开 `.ioc` 配置文件生成代码
2. **编译** — 使用 Keil 或 VS Code IDE 插件编译
3. **烧录** — 通过 ST-Link/SWD 下载到目标板
4. **传输音乐** — 通过 USB 大容量存储或读卡器放入 SD 卡

```
STM32CubeMX → 生成代码 → 打开 Keil/VS Code → 编译 → 烧录
```
